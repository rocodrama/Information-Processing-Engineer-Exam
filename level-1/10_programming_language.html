10. 프로그래밍 언어 활용

기본문법 · 언어특성 · 라이브러리 — 코드 출력 예측이 핵심!

C언어 Java Python 포인터 OOP 자료형

1 출제기준 요약

🔤 기본문법 활용하기

  • 데이터 타입, 변수 선언
  • 연산자 (산술, 관계, 논리, 비트)
  • 제어문 (if, switch), 반복문 (for, while)
  • 배열, 포인터, 구조체

🧬 언어특성 활용하기

  • C: 절차지향, 포인터, 메모리 직접 관리
  • Java: 객체지향, JVM, 예외처리
  • Python: 동적타입, 들여쓰기, GC
  • 클래스, 객체, 상속, 다형성

📚 라이브러리 활용하기

  • C: stdio.h, stdlib.h, string.h, math.h
  • Java: java.lang, java.util, java.io
  • Python: 표준 내장 함수 및 모듈
  • 라이브러리 검색·선택·구성·적용
⚠️ 시험 핵심: 코드를 읽고 출력 결과를 쓰는 문제가 매회 3~5문제 출제됩니다. C언어 포인터·증감연산자, Java 상속·오버라이딩, Python 슬라이싱이 특히 자주 나옵니다.

6 OOP 핵심 개념

🔒 캡슐화 (Encapsulation)

데이터+메서드를 하나로 묶고, private으로 외부 접근 제한

🧬 상속 (Inheritance)

부모 클래스의 속성·메서드를 자식이 물려받아 재사용

🎭 다형성 (Polymorphism)

구분 오버로딩 오버라이딩
개념 같은 이름, 매개변수 다름 부모 메서드 재정의
결정 컴파일 타임 런타임
반환타입 달라도 됨 같아야 함

🌫️ 추상화 (Abstraction)

구분 추상 클래스 인터페이스
선언 abstract class interface
구현 extends implements
다중상속 불가 가능
일반 메서드 가능 default만 가능
