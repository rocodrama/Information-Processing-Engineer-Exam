# 정보처리기사 실기 학습 콘텐츠 제작 플래너

## 기본 방향

- 목표: 정보처리기사 실기 출제기준을 기준으로, 개념 이해와 암기를 동시에 돕는 interactive HTML 학습 자료를 만든다.
- 기준 자료: `정보처리기사_실기_출제기준.pdf`
- 시험 정보: 필답형, 2시간 30분
- 적용기간: 2026.01.01 ~ 2026.12.31

## 제작 규칙

- 주요항목 12개를 각각 독립 HTML 파일로 만든다.
- 개념과 이해를 위한 interative html로 작성
- 단순 요약보다 시험 답안에 바로 쓸 수 있는 키워드와 문장형 정의를 우선한다.
- 절차, 계층, 비교, 흐름이 있는 내용은 애니메이션 또는 단계별 시각화로 만든다.
- SQL, 프로그래밍, 보안, 테스트는 실제 문제 풀이 중심으로 구성한다.

## 공통 HTML 목차 템플릿

```text
1. 출제기준 요약
2. 핵심 개념
3. 시험용 키워드
4. 절차/흐름/비교
5. 대표 문제 유형
6. 미니 퀴즈
7. 오답 노트
```
