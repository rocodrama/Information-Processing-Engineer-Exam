슬라이싱결과설명
lst[2:5][30, 40, 50]인덱스 2~4 (5 미포함)
lst[:4][10, 20, 30, 40]처음부터 3까지

클래스 상속 추적

 1class Animal:
 2    def __init__(self, name):
 3        self.name = name
 4    def speak(self):
 5        return f"{self.name}:..."
 6
 7class Dog(Animal):
 8    def __init__(self, name, breed):
 9        super().__init__(name)
10        self.breed = breed
11    def speak(self):           # override
12        return f"{self.name}({self.breed}):멍"
13
14a = Animal("고양이")
15d = Dog("바둑", "진돗개")
16print(a.speak())  # 고양이:...
17print(d.speak())  # 바둑(진돗개):멍
18print(isinstance(d, Animal))  # True
라인동작출력
14Animal("고양이") → self.name="고양이"
15Dog("바둑","진돗개") → super().__init__("바둑") → self.breed="진돗개"
16a.speak() → Animal의 speak()고양이:...
17d.speak() → Dog의 speak() (오버라이드)바둑(진돗개):멍

기출 빈출  다음 Python 코드의 출력 결과를 쓰시오. (클래스 상속 + super)
▼ 
class A:
    def __init__(self, x):
        self.x = x
    def calc(self):
        return self.x * 2

class B(A):
    def __init__(self, x, y):
        super().__init__(x)
        self.y = y
    def calc(self):
        return super().calc() + self.y

b = B(3, 10)
print(b.calc())
print(b.x, b.y)
정답
16
3 10
B(3,10): super().__init__(3) → self.x=3, self.y=10
b.calc(): super().calc() = A의 calc() = self.x*2 = 6
6 + self.y = 6+10 = 16

개념 확인  다음 Python 코드의 출력 결과를 쓰시오. (중첩 컴프리헨션)
▼ 
matrix = [[1,2,3], [4,5,6], [7,8,9]]

flat  = [x for row in matrix for x in row]
diag  = [matrix[i][i] for i in range(len(matrix))]
total = sum(flat)

print(flat[4])
print(diag)
print(total)
정답
5
[1, 5, 9]
45
flat = [1,2,3,4,5,6,7,8,9] → flat[4] = 5 (0-based)
diag: i=0→matrix[0][0]=1, i=1→matrix[1][1]=5, i=2→matrix[2][2]=9 → [1,5,9]
sum(1~9) = 45
개념 확인  Python의 mutable(가변) 타입과 immutable(불변) 타입을 각각 3가지씩 쓰시오.
▼
정답
mutable: list, dict, set
immutable: int, str, tuple
Mutable (변경 가능): list, dict, set — 내용을 수정해도 같은 객체 유지
Immutable (변경 불가): int, float, str, tuple, bool — 값 변경 시 새 객체 생성
⚠️ 함수에 list를 전달하면 원본 수정 가능! (mutable이라 참조 전달과 동일 효과)
